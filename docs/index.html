Bmore's Best Routes

Bmore's Best Routes shows you where Baltimore's most popular exercise areas are. You can compare different cities, neighborhoods, and even adjacent streets. The workouts include runs, swims, horseback rides, you name it. The purple numbers show the number of workouts posted for that region, which you can see by hovering over the number.

The data for this map was scraped from MapMyRun.com using the Selenium Webdriver in RStudio. Want to see how the map was made? Stay tuned for a Medium tutorial article, coming soon!
